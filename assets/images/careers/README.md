# Images pour la Page Carrières

## Instructions

Placez les 3 images suivantes dans ce dossier :

1. **office-space.jpg** - Photo de l'espace de bureau moderne (chaises ergonomiques, bureaux)
2. **developer-at-work.jpg** - Photo du développeur au travail (setup avec écrans multiples)
3. **office-lounge.jpg** - Photo de l'espace lounge/collaboration (zones de détente)

## Format Recommandé

- **Format** : JPG ou PNG
- **Résolution** : Minimum 1200x900px (ratio 4:3)
- **Taille** : Optimisé pour le web (< 500KB par image)
- **Qualité** : Haute qualité mais compressé

## Emplacement

Les images doivent être placées dans :
```
azzco-website/assets/images/careers/
```

## Noms de Fichiers Requis

- `office-space.jpg`
- `developer-at-work.jpg`
- `office-lounge.jpg`

Une fois les images ajoutées, elles apparaîtront automatiquement sur la page careers.html dans la section "Un Espace de Travail Moderne & Inspirant".
