# Images Artistiques IA-Humain

## Instructions

Placez les 2 images artistiques suivantes dans ce dossier :

1. **human-robot-connection.jpg** - Image symbolique de la main humaine et de la main robotique qui se touchent (fond magenta avec motif circuit board)
2. **neon-ai-connection.jpg** - Image néon de la collaboration humain-IA (fond cyan avec couleurs néon pink/green)

## Format Recommandé

- **Format** : JPG ou PNG
- **Résolution** : Minimum 1200x1200px (ratio 1:1 pour un meilleur rendu)
- **Taille** : Optimisé pour le web (< 800KB par image)
- **Qualité** : Haute qualité mais compressé

## Emplacement

Les images doivent être placées dans :
```
azzco-website/assets/images/ai-art/
```

## Noms de Fichiers Requis

- `human-robot-connection.jpg` - Utilisée dans le hero et la section "L'IA au Service de l'Humanité"
- `neon-ai-connection.jpg` - Utilisée dans la section "L'Intelligence Artificielle Réinventée"

## Utilisation

Ces images sont intégrées artistiquement dans :
1. **Hero Section** - En arrière-plan subtil (opacité 8%, grayscale)
2. **Section "L'IA au Service de l'Humanité"** - Image principale avec overlay doré
3. **Section "L'Intelligence Artificielle Réinventée"** - Image avec effet 3D et bordures dorées

Une fois les images ajoutées, elles apparaîtront automatiquement avec des effets artistiques (overlays dorés, filtres, animations).
